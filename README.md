# 🌍 International Education Costs Analysis

## 📖 Overview

This project focuses on data cleaning, preprocessing, and exploratory data analysis (EDA) of the **International Education Costs** dataset. The analysis includes graphical representation of both **univariate** and **bivariate** variables to understand patterns, trends, and relationships within the dataset.

The project was performed using Python libraries commonly used for data analysis and visualization.

---

## 🎯 Objectives

* ✨ Clean and preprocess the dataset
* 🧩 Handle missing and inconsistent values
* 📊 Perform exploratory data analysis (EDA)
* 📈 Analyze individual variables (Univariate Analysis)
* 📉 Analyze relationships between variables (Bivariate Analysis)
* 🎨 Create meaningful visualizations for better insights

---

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

---

## 📂 Dataset

**Dataset Name:** International_Education_Costs

The dataset contains information related to international education expenses such as tuition fees, living costs, accommodation, and other education-related financial factors.

---

## 🧹 Data Cleaning Performed

The following preprocessing steps were performed:

* Removed duplicate records
* Handled missing/null values
* Corrected inconsistent data entries
* Checked data types and converted columns where required
* Renamed columns for better readability
* Removed unnecessary columns (if applicable)

## 📊 Exploratory Data Analysis

### 📈 Univariate Analysis

Visualizations performed on individual columns:

* Histogram
* Count Plot
* Box Plot
* Distribution Plot
* Pie Chart

Purpose:

* Understand distribution of variables
* Detect outliers
* Analyze frequency and spread of data

---

### 📉 Bivariate Analysis

Visualizations performed to study relationships between two variables:

* Scatter Plot
* Bar Plot
* Heatmap
* Line Plot
* Correlation Analysis

Purpose:

* Identify trends and correlations
* Compare variables
* Understand relationships between education costs and other factors
